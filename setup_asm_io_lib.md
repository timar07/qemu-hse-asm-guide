# Исбользование библиотечных функций без SASM

## Для чего конктретно это нужно
Эта фигня нужна для того чтобы ипользовать фунцкии `io_print_dec`, `io_get_dec` итд

> Консольные комадны теперь без занака доллар ($) => теперь можно копировать всю строку

## Установочка
Заходим на виртуалку 

Скачиваем требуемые файлики (сама библиотека и скриптик для сборки)

```
wget http://asmcourse.cs.msu.ru/wp-content/uploads/2023/02/asm_1_3.zip
```

Эта команда скачивает вам архив

У вас должен быть установлен архиватор. Проверим это (если нет, первй запрос в гугле)

```
unzip --version
```
